# Comunicação e Recursos

## Canais de Comunicação

Se você tiver dúvidas, encontrar erros ou quiser sugerir melhorias, use os canais abaixo:

### Issues no GitHub

Abra uma [issue](../../issues) neste repositório para:

- Reportar erros no conteúdo
- Sugerir novos tópicos
- Tirar dúvidas sobre os exemplos
- Propor melhorias

Ao abrir uma issue, descreva claramente o problema ou sugestão e inclua o arquivo e a seção relacionados.

### Comunidade He4rt Developers

Este projeto faz parte da iniciativa **4Noobs** da [He4rt Developers](https://heartdevs.com/). Interaja com outros alunos e colaboradores na comunidade.
